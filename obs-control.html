OBS Setup (one-time)

1. Add obs-overlay.html as a Browser Source on your scene (transparent background, size it however you like).
2. Add obs-control.html as a second Browser Source anywhere (its visibility doesn't matter — you'll never see it on stream).
3. To make changes live: right-click the control source in OBS → Interact. That opens this panel in a live window — edit away, and the on-stream overlay updates instantly.
