Ice’s Bot Games Setup (Installation)
Download CS:GO here: https://store.steampowered.com/app/4465480/CounterStrikeGlobal_Offensive/
Download Bot Files (put in game): 
Go to your Steam Library
Open Properties… on Counter-Strike:Global Offensive App
Browse under Installed Files

Path should be (Drive):\Program Files (x86)\Steam\steamapps\common\csgo legacy
Open the “csgo” folder

Paste botprofile.db from the drive into this folder 
Locate resource folder

Paste csgo_english.txt file into the folder
Locate maps folder 
 Paste all the .nav files from the drive into it and replace the files.

Ice’s Bot Game Setup (In-Game)
Launch CS:GO
Choose Practice with Bots 
You can pick between these maps:
Inferno
Mirage
Anubis
Dust2
Tuscan
Cache
Vertigo

Ensure it’s Expert Bots and Set to Competitive Long Match
If first time setting commands join Spectator first
Open Game Settings Menu
Enable Developer Console (If needed rebind console key)
Join a team by pressing M to pick a team
Open the console it will look like this:
Paste these commands into the console: (make sure to send one line after another)

	bot_kick; mp_friendlyfire 0; mp_limitteams 32; mp_overtime_enable 1; mp_freezetime 6; mp_defuser_allocation 2; bot_defer_to_human_items 0;bot_defer_to_human_goals 0; bot_allow_grenades 0;
	mp_autoteambalance 0; mp_forcecamera 0; mp_match_restart_delay 50; mp_maxrounds 24; mp_ct_default_secondary weapon_usp_silencer; mp_c4timer 47; mp_damage_headshot_only 1
	mp_restartgame 1; (last command)
